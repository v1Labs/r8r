Basic Radar Chart

Simple single dataset visualization

const data = [ { label: 'Speed', value: 85 }, { label: 'Reliability', value: 92 }, { label: 'Comfort', value: 78 }, { label: 'Safety', value: 95 }, { label: 'Efficiency', value: 88 } ]; <R8R primaryData={data} />
